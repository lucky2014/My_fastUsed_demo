介绍

我们的框架是seajs+handlebars模块化开发。适合WAP和PC。我们的宗旨是把小项目的文件夹复制到任何地方都可以执行。

Seajs可以百度它的定义，以及用法，主要学习路径的配置。

Handlebars学习网站：http://www.cnblogs.com/iyangyuan/archive/2013/12/12/3471227.html。
